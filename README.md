# Proj Nexus V1

An AI-powered desktop productivity assistant built using Python that combines automation, voice interaction, OCR utilities, clipboard intelligence, and local workflow tools into a single system.

## Features

* Voice-based assistant interaction
* Clipboard intelligence and smart commands
* OCR-based utilities
* Reminder and note management
* Local desktop automation
* Executable desktop build using PyInstaller
* Modular assistant architecture

## Tech Stack

* Python
* Tkinter
* SpeechRecognition
* pyttsx3
* PyInstaller
* OCR libraries
* JSON-based storage

## Installation

Clone the repository:

```bash
git clone https://github.com/Satyarth740/proj-nexus-v1.git
cd proj-nexus-v1
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the assistant:

```bash
python main.py
```
